# Contributing to Industrial Robot Simulation

Thank you for your interest in contributing to this project!

## How to Contribute

1.  **Fork the Repository**: Create your own fork of the project.
2.  **Clone the Fork**: Clone your fork to your local machine.
3.  **Create a Branch**: Create a new branch for your feature or bug fix.
4.  **Make Changes**: Implement your changes and ensure they follow the project's coding style.
5.  **Add Tests**: If you're adding a new feature, please include corresponding unit tests in the `tests/` directory.
6.  **Run Tests**: Ensure all tests pass before submitting.
7.  **Submit a Pull Request**: Submit a PR to the `main` branch of the original repository.

## Coding Standards

- Follow PEP 8 for Python code.
- Ensure all mathematical implementations are documented in the code or README.
- Keep the visualization clean and responsive.

## Questions?

Feel free to open an issue for any questions or suggestions.
